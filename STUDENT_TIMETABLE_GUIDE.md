# Student Timetable Feature Guide

## Overview
Students can now view their generated class timetable based on their department and semester. The timetable displays all scheduled classes with course details, faculty information, room assignments, and time slots.

## Features

### 1. Dashboard View
- Quick overview of your class schedule
- Statistics showing enrolled courses, classes today, free periods, and total credits
- "View Full Schedule" button to access the dedicated timetable page

### 2. Dedicated Timetable Page
Access via: **Sidebar → My Timetable** or **Dashboard → View Full Schedule**

#### Key Features:
- **Complete Weekly Schedule**: Full grid view of all classes organized by day and time
- **Statistics Cards**: 
  - Total Classes per week
  - Number of enrolled courses
  - Number of faculty instructors
  - Number of classroom locations
- **Visual Grid**: Color-coded timetable showing:
  - Course code and name
  - Faculty name
  - Room/classroom
  - Class type (Lecture/Lab/Seminar)
- **Export Options**:
  - Download as PDF
  - Print directly from browser

### 3. Automatic Data Loading
- Timetable automatically loads based on your profile (department & semester)
- Real-time updates when timetable is regenerated by admin
- Error handling with retry option if loading fails

## How to Use

### Viewing Your Timetable

1. **Login** to your student account
2. Navigate to **"My Timetable"** from the sidebar
3. Your timetable will automatically load based on your:
   - Department (e.g., Computer Science)
   - Semester (e.g., Semester 3)

### Understanding the Timetable Grid

Each cell in the timetable contains:
- **Course Code/Name**: The subject you're enrolled in
- **Faculty Name**: Your instructor
- **Room**: Classroom location
- **Type Badge**: Lecture, Lab, or Seminar

Color coding:
- 🔵 Blue border: Lecture
- 🟢 Green border: Lab
- 🟡 Yellow border: Seminar

### Downloading/Printing

1. Click **"Download PDF"** button to save a copy
2. Click **"Print"** to print directly
3. Use browser's print dialog for additional options

## What If No Timetable Is Available?

If you see "No Timetable Available":
- The timetable for your department/semester hasn't been generated yet
- Contact your administrator or academic office
- Check back later after timetable generation

## Technical Details

### API Endpoint
```
GET /api/timetable/:department/:semester
```

### Data Structure
The timetable includes:
- Course information (name, code)
- Faculty details (name, email)
- Room assignments (name, capacity)
- Time slots (day, start time, end time)

### Automatic Updates
- Timetable data is fetched from the backend
- Uses React Query for caching and automatic refetching
- Updates reflect immediately after admin regenerates timetable

## Troubleshooting

### Timetable Not Loading
1. Check your internet connection
2. Verify you're logged in
3. Click the "Retry" button
4. Refresh the page
5. Contact support if issue persists

### Wrong Timetable Displayed
- Verify your profile has correct department and semester
- Contact admin to update your profile if needed

### Missing Classes
- Some classes may not be scheduled yet
- Contact your department if you notice missing courses
